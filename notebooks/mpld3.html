MPLD3¶

mpld3 conceptually works as a backend for matplotlib. You make your figures with matplotlib commands that you probably know already and then save it as a html website with mpld3.

mpld3 is stable - it's not under active development and no new features are added, but the maintainers update it regularly to make sure it keeps working with new version of numpy ans matplotlib.

If you already know matplotlib, mpld3 is a great way to get started with interactive plots; though not all (but most) matplotlib features are supported.

import matplotlib.pyplot as plt
import numpy as np
import mpld3
from mpld3 import plugins

Here, I'm using a table with data from one of my old papers. We want to compare optical reddening and X-ray absorption. There are going to be a lot of stars and sometimes we may want to know more about each star, but we also don't have the space to label each and every point in our plot or the names would all overlap. This is a great use of an interactive figure! Just use your mouse to identify the stars you're interested in.
